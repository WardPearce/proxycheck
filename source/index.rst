Welcome to ProxyCheck.io's unofficial documentation!
====================================================

This is a unofficial asynchronous & synchronous wrapper for ProxyCheck.io API.

**Features:**

- Full API coverage.
- Asynchronous & synchronous support.
- Easy to use with an object oriented design.
- 100% test coverage.

Install
-------

Pip: ::

      pip3 install proxycheck

Git: ::

      pip3 install git+https://github.com/WardPearce/proxycheck.git
